Watch Your Words Travel Through an LLM

Type a phrase and follow it step-by-step through tokenization, embeddings, attention, the forward pass, and next-token prediction.

Try:
1

Tokenize

Breaking text into pieces the model can understand

Your phrase is split into tokens — subword units from the model's vocabulary. Each token gets a numeric ID.

BPE (Byte Pair Encoding) iteratively merges the most frequent character pairs. The vocabulary (~32K–100K tokens) balances between character-level and word-level representations. Common words are single tokens; rare words split into subwords.

↓
2

Embed

Converting tokens into meaning-vectors

Each token becomes a high-dimensional vector that captures its meaning. Similar words land near each other in this space.

Each token ID indexes into a learned embedding matrix E ∈ ℝV×d. Real models use d=768–12288 dimensions. The vectors encode semantic and syntactic properties through training on massive text corpora.

3

Attention

Tokens decide which other tokens matter

Each token looks at every other token and assigns an attention weight — how relevant is that token for understanding me?

Attention(Q,K,V) = softmax(QKT/√dk)V. Queries, Keys, and Values are linear projections. Multiple heads (typically 8–96) attend to different relationship types simultaneously.

4

Forward Pass

Data flows through the transformer architecture

Your tokens flow through layers of the transformer — each layer refines the representation using attention and feed-forward networks.

Each layer: x → MultiHeadAttention(x) + x → LayerNorm → FFN(x) + x → LayerNorm. The residual connections ("+x") help gradients flow. Real models stack 12–96 of these layers.

5

Predict Next Token

What comes next?

The model outputs a probability for every possible next token. Temperature and top-p control how "creative" the sampling is.

Final hidden states project to vocabulary-sized logits via Wout. Temperature T scales logits: P(i) = exp(zi/T) / Σ exp(zj/T). Top-p (nucleus) sampling truncates to smallest set with cumulative probability ≥ p.
